# Task5_Pandas_Cleaning
Python data cleaning using Pandas on the Titanic dataset (handling missing values, duplicates, datatype conversion, and feature creation).

Dataset: Titanic Dataset

### Tasks Performed:
- Loaded dataset using pandas
- Checked structure using head() and info()
- Identified missing values
- Filled missing Age values using median
- Dropped Cabin column due to excessive missing values
- Filled Embarked using mode
- Removed duplicate rows
- Converted Survived and Pclass to categorical types
- Created a new column (Age_Group)
- Saved cleaned dataset as CSV

### Tools Used:
- Python
- Pandas
- Google Colab
